

## Prerequisites

- [Node.js](https://nodejs.org/)

- Clone this repository in your machine. (via command git clone or download)


## Install

```sh
npm install
```

## Run

Open Cypress:

```sh
npx cypress open
```
## Run Cypress in command line:
Run Cypress in command line:

```sh
npx cypress run
```

## Run Cypress in chrome headless

Run Cypress in chrome:

```sh
npm run cypress:run:chrome
```
